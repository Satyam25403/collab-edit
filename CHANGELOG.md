# Changelog

## [1.0.0] - 2026-06-20

### Added
- Real-time collaborative editing between two VS Code instances
- 6-digit room code system for host/guest pairing
- Virtual filesystem (`collabfs://`) for guests to browse host workspace
- Read-only mode by default with permission request flow
- Host approval/denial of guest edit permissions
- Live cursor and text sync via WebSocket relay
- Sidebar panel with connection status, room code display, and session controls
- Leave session / disconnect handling
